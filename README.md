# 链家网新房数据可视化及预测系统 python715

## 项目概述

本项目是基于Python语言及Flask框架开发的房产数据管理系统。系统通过爬虫技术获取链家网全国新房数据，并进行数据清洗、存储、可视化和预测分析。

## 技术栈

- **爬虫技术**：requests
- **数据库**：MySQL
- **后端框架**：Flask
- **机器学习算法**：scikit-learn
- **可视化工具**：Echarts

## 功能模块

- 数据爬取：自动抓取链家网新房数据
- 数据存储：使用MySQL存储爬取的房产信息
- 数据可视化：利用Echarts进行数据可视化展示
- 预测分析：运用多元线性回归算法进行房价预测

### 爬取字段

- 房名
- 封面
- 城市/区域
- 地区
- 详细地址
- 房型
- 建筑面积
- 预售证情况
- 装修情况
- 开发公司
- 房屋类型
- 交房时间
- 开盘时间
- 标签
- 总价区间
- 售房情况
- 详情链接

## 系统角色

- 系统管理员
- 数据分析师

## 运行环境

- Python 3.x
- Flask
- MySQL
- 相关Python库（如requests, scikit-learn等）

## 部署步骤

1. 环境配置：安装Python及相关的依赖库
2. 数据库搭建：创建MySQL数据库和表结构
3. 爬虫部署：运行爬虫脚本，开始数据爬取
